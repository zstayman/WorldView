# GA WDI NYC, Project 1
WorldView visualizes the last two days of New York Times Articles and displays them on a map.

### Libraries, Gems, and APIs
1. [Redis-rb](https://github.com/redis/redis-rb)
2. [Redis Expiring Sets](https://github.com/wellbredgrapefruit/redis-expiring-set)
3. [Mapbox](https://www.mapbox.com/)
4. [Heroku Scheduler](https://devcenter.heroku.com/articles/scheduler)
